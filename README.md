# OSCP PEN-200 Study Guide & Labs

A complete study guide and hands-on lab walkthroughs for the **Offensive Security Certified Professional (OSCP)** certification, based on the **PEN-200: Penetration Testing with Kali Linux** course.

This repository is built for learners preparing for the OSCP / OSCP+ exam. Every lab is documented step-by-step with screenshots so you can follow along, replicate the work, and understand the *why* behind each command — not just copy and paste.

## About OSCP PEN-200

The OSCP is one of the most respected hands-on penetration testing certifications in the industry. Unlike multiple-choice exams, it is awarded based on a **24-hour practical exam** in which the candidate must compromise multiple machines and submit a professional penetration test report.

PEN-200 is the official course that prepares you for this exam. It covers:

- Penetration testing methodology
- Information gathering and enumeration
- Web application attacks
- Client-side attacks
- Public exploit usage and modification
- Privilege escalation (Linux & Windows)
- Password attacks
- Pivoting and tunneling
- Active Directory exploitation
- Professional report writing

A passing score is **70 / 100**, with significant weight given to the Active Directory set.

## How to Use This Repository

1. Read the chapter in the study guide first to understand the theory.
2. Open the matching lab walkthrough folder.
3. Recreate each step in your own Kali Linux VM — **do not just read**. The OSCP rewards muscle memory.
4. After completing a chapter, attempt similar machines on platforms like HackTheBox, TryHackMe, or PG Practice to reinforce the skills.

## Recommended Prerequisites

Before diving in, you should be comfortable with:

- Basic scripting (Bash and/or Python)
